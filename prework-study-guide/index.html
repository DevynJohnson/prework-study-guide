HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The details in HTML are important for those who nagivate webpages with adaptive tools
  • ul creates a list, with li in between to create line items/bullet points

CSS

  • CSS is used to stylize and format the HTML content on a webpage
  • There are three ways to style a webpage with CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • External style sheets are common as it allows saving CSS and HTML codes on different files
  • External style sheets are linked to the HTML document using a link function in the HTML code
  • A CSS rule is comprised of a selector (which defines the elements affected) and a declaration, containing a property and value to be applied to those elements
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls changes from main branch to merge with local branch
  • git commit -m "descripton": creats a commit, explaining and signing the changes made
  • git push origin branch name: push local branch code to the main branch and create pull request
