CSS

  • Enter your CSS notes here
  • CSS is used to stylize and format the HTML content on a webpage
  • There are three ways to style a webpage with CSS: inline CSS, internal CSS style sheet, and external CSS style sheet
  • External style sheets are common as it allows saving CSS and HTML codes on different files
  • External style sheets are linked to the HTML document using a link function in the HTML code
  • A CSS rule is comprised of a selector (which defines the elements affected) and a declaration, containing a property and value to be applied to those elements
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pulls changes from main branch to merge with local branch
  • git commit -m "descripton": creats a commit, explaining and signing the changes made
  • git push origin branch name: push local branch code to the main branch and create pull request

JavaScript

  • Enter your JavaScript notes here
  • JavaScript is a high-level, dynamic programming language that allows users to ineract with webpages
  • JS is linked to your webpage using a script element in the HTML code, much like the link element to apply a CSS Style Sheet
  • Script element should be added at the bottom of your HTML code, it may cause issues if it loads before any HTML content that it is supposed to interact with because the web browser reads code in order
  • JS and many other programming languages are zero-indexed and sequential, meaning index values begin at 0 and indicies are whole numbers that increase by 1 each time
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
